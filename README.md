# 🏸 Badminton Club Management System
**แอปพลิเคชันจัดการสโมสรแบดมินตัน Snowite**

เป็นแอปพลิเคชันเว็บสำหรับจัดการข้อมูลสมาชิก แมตช์ สถิติ และการเงินของสโมสรแบดมินตัน

## 🎯 Features

### 👥 **จัดการสมาชิก**
- ✅ เพิ่มสมาชิกใหม่ (ชื่อ, เบอร์โทร)
- ✅ แก้ไขชื่อและเบอร์โทรสมาชิก
- ✅ ลบสมาชิก (ต้องยืนยันด้วยคำนวณเลข 2 ขั้น)
- ✅ เลือก icon 30 แบบให้สมาชิก:
  - 👥 ทั่วไป: 👤🧑👨👩🧔👨‍🦰👩‍🦰🧑‍🦱👨‍🦱👩‍🦱
  - 🎌 ญี่ปุ่น/อนิเมะ: 👘🌸⛩️🍜🍙🥋🎎🎏🎐🎭
  - 🤪 กวนๆ ตลกๆ: 🤡🤣😜🤪😝🎪🚀💥🤖👽

### 📊 **บันทึกแมตช์และสถิติ**
- ✅ บันทึกผลแมตช์ (ผู้ชนะ/ผู้แพ้)
- ✅ ระบบ ELO โดยอัติโนมัติ (ปรับคะแนนหลังแต่ละแมตช์)
- ✅ ติดตามสัญญาว่า Freeze streak หลังจากแพ้ 3 ครั้งติดต่อ
- ✅ สถิติเชิงลึก:
  - **Ranking**: อันดับสมาชิกตามคะแนน ELO พร้อมแจก 🥇🥈🥉
  - **Win Rate**: เปอร์เซ็นต์การชนะ
  - **Form**: สถิติ 5 แมตช์ล่าสุด
  - **Head-to-Head**: ประวัติเกมระหว่างสมาชิกแต่ละคู่
  - **Streaks**: ติดตามจำนวนแพ้ติดต่อ (Freeze 🧊 ตั้งแต่ 3 ครั้ง)

### 💰 **จัดการการเงิน**
- ✅ บันทึกค่าสนาม (ชั่วโมง × ราคา)
- ✅ บันทึกค่าลูกแบดมินตัน (จำนวน × ราคา) สำหรับผู้ใช้สูงสุด 3 คน
- ✅ แบ่งค่าใช้จ่ายอัติโนมัติ (equal split)
- ✅ ติดตามสถานะการจ่าย (ผู้ชำระแล้ว/ยังไม่ชำระ)
- ✅ ประวัติค่าใช้จ่าย (Create, Read, Update, Delete)

### 🎨 **UI/UX**
- ✅ Dark/Light theme toggle 🌙/☀️
- ✅ Firebase Realtime Database sync status 🟢🔴
- ✅ Toast notifications ทั้งภาษาไทยและอีโมจิ
- ✅ Responsive design (ใช้งานได้บนมือถือ, แท็บเล็ต, เดสก์ทอป)

## 🔧 **เทคโนโลยี**

### Frontend
- **HTML/CSS/JavaScript** (Vanilla - ไม่มี framework)
- **Single Page Application (SPA)** - ทั้งแอปอยู่ไฟล์เดียว `index.html`

### Backend & Database
- **Firebase Realtime Database** - บันทึกข้อมูลแบบ Real-time
- **Firebase Analytics** - ติดตามการใช้งาน
- **localStorage** - เก็บข้อมูลเพื่อใช้ offline หรือโหลดเร็ว

### Deployment
- **GitHub Pages** - Host บน GitHub อย่างฟรี

## 🚀 **วิธีใช้งาน**

### เข้าใช้แอป
```
https://badminton_snowite.github.io (ผ่าน GitHub Pages)
หรือ
เปิด index.html ใน browser
```

### 1️⃣ เพิ่มสมาชิก
1. ไปที่แท็บ **👥 สมาชิก**
2. กดปุ่ม **➕ เพิ่มสมาชิก**
3. กรอกชื่อ (บังคับ) และเบอร์โทร (ไม่บังคับ)
4. กดปุ่ม **✅ เพิ่ม**

### 2️⃣ เลือก Icon สมาชิก
1. ไปที่แท็บ **📊 สถิติ**
2. คลิกสมาชิกใดสมาชิกหนึ่ง
3. กดปุ่ม **🎨 เลือก icon**
4. เลือก icon ที่ชอบ (จะบันทึกโดยอัติโนมัติ)

### 3️⃣ บันทึกแมตช์
1. ไปที่แท็บ **🎯 แมตช์**
2. กดปุ่ม **➕ เพิ่มแมตช์**
3. เลือก ผู้ชนะ (2 คน) และ ผู้แพ้ (2 คน)
4. กรอก คะแนน (ไม่บังคับ) และ ตัวเลือกอื่น
5. กดปุ่ม **✅ บันทึก**

### 4️⃣ บันทึกค่าใช้จ่าย
1. ไปที่แท็บ **💰 การเงิน**
2. กดปุ่ม **➕ เพิ่ม session**
3. กรอก:
   - วันที่
   - ผู้จอง/ผู้จ่าย
   - จำนวนชั่วโมง × ราคา/ชม.
   - ผู้ร่วมตี (เลือกหลายคน)
   - ค่าลูกแบดฯ (ไม่บังคับ)
4. กดปุ่ม **✅ บันทึก**

## 💾 **การบันทึกข้อมูล**

- ✅ **Auto-sync**: ข้อมูลทั้งหมดบันทึกไป Firebase อัติโนมัติ
- ✅ **localStorage**: ยังเก็บใน localStorage เพื่อใช้ offline ได้
- ✅ **Firebase status**: มองได้ที่ 🟢 (online) / 🔴 (offline) ที่ header

## 🔐 **ความปลอดภัย**

⚠️ **สำคัญ**: Firebase Realtime Database อยู่ในโหมด **สาธารณะ** (public read/write)
- ใครก็ตามที่หา URL ของ database ได้ สามารถอ่าน/แก้ไข/ลบข้อมูลได้
- ถ้าต้องการความปลอดภัย ให้เซต Firebase Security Rules เป็นแบบ private
